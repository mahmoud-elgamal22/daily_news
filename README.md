# 📰 Daily News

A simple and elegant news application built with Flutter that allows users to stay updated with the latest news. 🌍

## 📋 Features

- **Latest News**: Get real-time updates from various news sources. 📰
- **Web View**: Read full articles within the app using a built-in web view. 🌐
- **Swipe to Refresh**: Refresh the news feed with a simple swipe gesture. 🔄
- **Cached Images**: Fast loading with cached images for better performance. 📸
- **Splash Screen**: A beautiful splash screen that greets users when they open the app. 🎨

## 🚀 Getting Started

### Prerequisites

- Flutter SDK version `>=3.4.4 <4.0.0`
- An IDE such as Android Studio, Visual Studio Code, or IntelliJ IDEA.
  
🛠️ Usage
Launch the app to see the latest news articles. 📖
Tap on any article to read more details in a web view. 🌐
The splash screen will be displayed while the app is loading. 🌈
Pull down to refresh the news feed. 🔄

### 📦 Dependencies

The following dependencies are used in this project:

- `flutter`: Core Flutter SDK.
- `cupertino_icons`: Icons for iOS-style apps.
- `flutter_native_splash`: For implementing splash screens.
- `dio`: A powerful HTTP client for Dart.
- `cached_network_image`: Caching images for improved performance.
- `webview_flutter`: To display web content within the app.
